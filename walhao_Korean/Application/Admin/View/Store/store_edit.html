1. 可以修改自营店铺的店铺名称以及店铺状态是否为开启状态

2. 可以修改自营店铺的店主商家中心登录账号。

3. 如需修改店主登录密码，请到会员管理中，搜索“店主账号”相应的会员并编辑。

4. 已绑定所有类目的自营店，如果将“绑定所有类目”设置为“否”，则会下架其所有商品，请谨慎操作！

1. Can modify the name of the shop and the store is open state

2. Can modify the owner of the store owner shop login account.

3. If you want to modify the owner login password, please go to the member management, search the owner account number of the corresponding members and editors.

4. All the categories of the store has been bound, if the "binding all categories" set to "no", it will be on the shelves of all its goods, please be careful!
